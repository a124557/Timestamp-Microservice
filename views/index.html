Timestamp Microservice

About

A full stack microservice developed with JavaScript and Node.js. Takes a URL ending in a date with a standard format (e.g. 2015-10-12) or in milliseconds (e.g 1451001600000), and returns the date and time in a unix and utc key within a json object.

Examples

https://boilerplate-project-timestamp-1.av1124.repl.co/api/2015-10-12
https://boilerplate-project-timestamp-1.av1124.repl.co/api/1451001600000

Sample Output

{"unix":1444608000000,"utc":"Mon, 12 Oct 2015 00:00:00 GMT"}
